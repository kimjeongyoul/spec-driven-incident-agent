### 토큰이란

&nbsp;

```
영어: 1 단어 ≈ 1.3 토큰
  "The quick brown fox jumps" → 5 단어 → ~6 토큰

한글: 1 글자 ≈ 2~3 토큰
  "안녕하세요" → 5 글자 → ~12 토큰

코드: 줄마다 다르지만, 코드 1줄 ≈ 10~20 토큰
  "const x = await fetch('/api/data');" → ~12 토큰
```

## 5. 온프레미스 vs 클라우드 API: 손익분기점

&nbsp;

### 온프레미스 비용 구조

&nbsp;

```
[초기 투자]
- GPU 서버 (A100 80GB × 2): ~$30,000 (4,290만원)
- 서버 본체 (CPU, RAM, SSD): ~$5,000 (715만원)
- 네트워크/랙: ~$2,000 (286만원)
- 초기 총합: ~$37,000 (5,291만원)

[월간 운영비]
- 전기료: ~$200/월 (29만원) — GPU 서버 24시간 가동
- 인터넷/네트워크: ~$50/월 (7.2만원)
- 유지보수 인력: 포함 안 함 (기존 인프라팀 활용 가정)
- 월간 총합: ~$250/월 (36만원)

[GPU 감가상각]
- 3년 감가: $37,000 / 36개월 = $1,028/월 (147만원)
- 월간 실질 비용: $1,028 + $250 = $1,278/월 (183만원)
```

&nbsp;

### 손익분기점 계산

&nbsp;

```
클라우드 API 월 비용이 $X일 때:

$X < $250/월   → 클라우드 API가 무조건 유리
$X = $1,278/월 → 온프레미스와 동일 (3년 기준)
$X > $1,500/월 → 온프레미스 검토 가치 있음
$X > $3,000/월 → 온프레미스가 확실히 유리

시각화:
  
  비용
  ($/월)
    │
  3K├──────────────────╱── 클라우드 API
    │               ╱
  2K├────────────╱
    │         ╱
  1.3K├──────╳──────────────── 온프레미스 (감가 포함)
    │     ╱
  1K├──╱
    │╱
    └────────────────────────── 사용량
      10M   50M   100M   500M  (월간 토큰)
```

&nbsp;

### 판단 기준

&nbsp;

```
월 API 비용 $500 미만     → 클라우드 API (고민할 필요 없음)
월 API 비용 $500~$1,500   → 클라우드 API (운영 편의성 우세)
월 API 비용 $1,500~$3,000 → 하이브리드 검토
월 API 비용 $3,000 이상   → 온프레미스 적극 검토

단, 온프레미스의 숨은 비용을 고려해야 함:
- GPU 운영 인력 (없으면 학습 비용)
- 모델 업데이트 관리
- 장애 대응
- 냉각/전력 인프라
```

### 6-1. 프롬프트 캐싱

&nbsp;

같은 시스템 프롬프트를 반복 호출할 때 캐싱을 사용하면 비용이 크게 줄어든다.

&nbsp;

```typescript
// Anthropic 프롬프트 캐싱
const response = await anthropic.messages.create({
  model: 'claude-sonnet-4-20250514',
  max_tokens: 1024,
  system: [
    {
      type: 'text',
      text: longSystemPrompt,  // 긴 시스템 프롬프트
      cache_control: { type: 'ephemeral' },  // 캐싱 활성화
    },
  ],
  messages: [{ role: 'user', content: userMessage }],
});

// 캐시 히트 시 입력 비용 90% 절약
// Claude Sonnet: $3/M → $0.30/M (캐시 히트)
```

&nbsp;

```
프롬프트 캐싱 효과 예시:

시스템 프롬프트: 2,000 토큰 (모든 요청에 동일)
사용자 메시지: 500 토큰 (매번 다름)

캐싱 없이 100회 호출:
- 입력: (2,000 + 500) × 100 = 250,000 토큰

캐싱 적용 100회 호출:
- 첫 호출: 2,500 토큰 (정가)
- 나머지 99회: 500 × 99 = 49,500 토큰 (정가) + 2,000 × 99 (캐시 = 90% 할인)
- 실질 비용: 약 40% 절감
```

### 6-2. 작은 모델로 전처리, 큰 모델은 핵심만

&nbsp;

```typescript
// 2단계 처리: 작은 모델 → 큰 모델
async function processRequest(message: string): Promise<string> {
  // Step 1: 작은 모델로 분류 + 필터링 (저렴)
  const classification = await callCheapModel(message);
  // GPT-4o mini: $0.15/M 입력
  
  if (classification.category === 'simple') {
    // 단순 질문은 작은 모델로 처리
    return await callCheapModel(message);
  }
  
  // Step 2: 복잡한 질문만 큰 모델로 처리 (비싸지만 정확)
  return await callExpensiveModel(message);
  // Claude Sonnet: $3/M 입력
}

// 전체 요청 중 70%가 단순 → 70% 비용 절감
```

### 6-4. 비용 모니터링 + 알림

&nbsp;

```typescript
// 비용 추적 시스템
class CostTracker {
  private dailyCost = 0;
  private monthlyCost = 0;

  track(usage: { inputTokens: number; outputTokens: number; model: string }) {
    const cost = this.calculateCost(usage);
    this.dailyCost += cost;
    this.monthlyCost += cost;

    // 일 예산 초과 시 알림
    if (this.dailyCost > DAILY_BUDGET) {
      notify(`일 예산 초과: $${this.dailyCost.toFixed(2)} / $${DAILY_BUDGET}`);
    }

    // 월 예산 80% 도달 시 경고
    if (this.monthlyCost > MONTHLY_BUDGET * 0.8) {
      notify(`월 예산 80% 도달: $${this.monthlyCost.toFixed(2)}`);
    }
  }

  calculateCost(usage: any): number {
    const rates = {
      'gpt-4o': { input: 2.5, output: 10 },
      'claude-sonnet-4-20250514': { input: 3, output: 15 },
      'gpt-4o-mini': { input: 0.15, output: 0.6 },
    };
    const rate = rates[usage.model];
    return (usage.inputTokens * rate.input + usage.outputTokens * rate.output) / 1_000_000;
  }
}
```
